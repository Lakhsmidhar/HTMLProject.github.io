FrontEnd Development

HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML is the backbone of website design and function. It provides a structure for web pages that allows browsers to understand and display the content on the page.By understanding how HTML works, you can create better websites with more sophisticated features


CSS

CSS is made up of coding elements and "cascading style sheets". These style sheets are also called CSS files. CSS describes how HTML elements should be displayed on screen, paper, or in other media. It can control the layout of multiple web pages at once and save a lot of work. Before CSS, tags like font, color, background style, element alignments, border, and size had to be repeated on every web page. CSS solves this problem by having an inherent hierarchy. The "cascading" in CSS refers to the fact that styling rules "cascade" down from several sources. This means that CSS has an inherent hierarchy and styles of a higher precedence will overwrite rules of a lower precedence.


JavaScript

JavaScript is a high-level, interpreted programming language that is used to make web pages interactive. It is one of the three core technologies of World Wide Web content production (along with HTML and CSS). JavaScript is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive. It is used to make web pages interactive and add dynamic content to web pages. It is also used to create web applications and games.


BackEnd Development

Java

Java is a general-purpose, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is a general-purpose programming language intended to let programmers write once, run anywhere (WORA), meaning that compiled Java code can run on all platforms that support Java without the need to recompile. Java is used to develop a wide variety of applications, including mobile apps, desktop apps, web apps, and enterprise software. It is also a popular language for teaching programming.


Spring Framework

Spring Framework is a popular Java framework that provides a comprehensive infrastructure for building web applications. It includes features such as dependency injection, transaction management, and web MVC.


Hibernate

Hibernate is an object-relational mapping (ORM) library that provides a way to map Java objects to database tables. It is used to perform database operations without having to write SQL code.


Servlets and JSPs

Servlets and JSPs are Java technologies that are used to build web applications. Servlets are Java classes that handle HTTP requests and responses. JSPs are JavaServer Pages that are used to generate dynamic web content.



DataBase

MYSql

MySQL is a relational database management system (RDBMS). It is a popular, open-source SQL database management system (DBMS) developed by Oracle Corporation. MySQL is based on structured query language (SQL). MySQL stores data in separate tables and organizes the database structure into physical files optimized for speed. The logical data model offers a flexible programming environment with objects such as data tables, views, rows, and columns.
